## Electricity price forecast (15-min)

This project predicts 15-minute RCE electricity prices using machine learning.

### How to run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt

2	Train & evaluate models (also saves the best model bundle):
  python price_of_electricity_en.py
3.	Deployment demo (load saved model and predict):
  python deploy_predict_en.py
